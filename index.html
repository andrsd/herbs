Herbs

Monitors and controls an indoor herb garden

Download this project as a .zip file Download this project as a tar.gz file

This page describes a small project that allows me to monitor herbs that I grow indoors. I am basically interested in the moisture level for each herb. Additionally I can monitor temperature and humidity. The collected information is then uploaded into separate feeds on io.adafruit.com. Feeds are monitored by IFTTT application and when the moisture level falls below specified value, a notification is send to a smart phone, i.e. time to water the herbs.
